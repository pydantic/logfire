# Python SDK

The Python SDK provides the deepest Logfire integration: manual spans and logs, metrics, auto-instrumentation, framework integrations, Python object rendering, testing helpers, and a generated API reference.

## Where to start

- [Getting Started](get-started.md) installs the SDK, configures a project, and sends your first telemetry.
- [Instrumentation](instrumentation/index.md) covers manual spans, auto-tracing, metrics, distributed tracing, sampling, scrubbing, and suppression.
- [Integrations](integrations/index.md) covers Python web frameworks, AI and LLM libraries, databases, HTTP clients, task queues, logging, testing, and other packages.
- [Configuration](reference/configuration.md) documents environment variables, `pyproject.toml`, and `logfire.configure()`.
- [API Reference](reference/api/logfire.md) documents the Python package surface.

## Platform docs

SDK setup sends telemetry into the Logfire platform. Use the platform sections for [Observe & Investigate](../guides/web-ui/live.md), [Evaluate](../guides/web-ui/evals.md), [Manage & Configure](../guides/web-ui/organizations-and-projects.md), and [Deploy & Scale](../enterprise.md).
